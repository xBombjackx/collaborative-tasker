      Command Guide

      Viewer Commands:

      • !task [description] - Submit a task for approval.
      • !status [complete|pause|resume] - Update your task status.
      • !help - Shows this guide.

      Moderator/Streamer Commands:

      • !approve [username] - Approve a pending task.
      • !reject [username] - Reject a pending task.
      • !status [complete|pause|resume] [username] - Update a user's task.
      • !addtask [list_name] [description] - Add a task to a streamer list.
