Available Commands

  • !task [description]Submits a new task to the default 'Viewers' list for approval.
  • !addtask [list] [desc]Adds a task directly to a specified list.
  • !status [complete|pause]Updates the status of your own active task.
  • !approve [username]MOD: Approves a user's pending task.
  • !reject [username]MOD: Rejects a user's pending task.
  • !donetask [username]MOD: Marks a user's active task as complete.
